# CPRG306 Assignments

## Running the Project
1. Install the required packages.
```bash
yarn install
```

2. Run the app on your local machine.
```bash
yarn dev -p 3000 ## You can change the port number.
```

3. Navigate to [http://localhost:3000](http://localhost:3000) in your browser to see the app.
